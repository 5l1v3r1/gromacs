gcp file format

Files with the gcp file extension contain input data for the genconf program.

A sample is included below.

;
;	User spoel (20016)
;	Thu Nov 24 11:04:49 1994
;	Input file
;
nx                  =  6	; Number of copies in X dir
ny                  =  3	; Id. Y
nz                  =  6	; Id. Z
dx                  =  0.3	; Extra distance in X dir between grid points
dy                  =  0.15	; Id. Y
dz                  =  0.15	; Id. Z
seed                =  1993	; Random number generator seed
