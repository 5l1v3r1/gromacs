XVGR, The ultimate plotting program

Almost all output from Gromacs analysis tools is ready as input for xvgr. We use xvgr, because it is very flexible, and it is also free software. It produces PostScript(tm) output, which is very suitable for inclusion in eg. LaTeX documents, but also for other word processors.
