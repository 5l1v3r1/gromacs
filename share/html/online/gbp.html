gbp file format

Files with the gbp file extension contain input data for the genbox program.

A sample is included below.

;
;	Genbox Input file
;
addprotein          =  yes	; yes, no 
addsolvent          =  yes	; yes, no
nmol                =  0	; number of extra molecules to be added
replace_dist        =  0.105	; default vdwaal radius for atoms not in
				; in database
box                 =  cubic	; cubic (=rectangular) trunc (truncated 
				; octahedron)
Beta                =  0.0	; Obsolete
box_x               =  0.6	; Extra space around solute X
box_y               =  0.6	; Id. y
box_z               =  0.6	; Id. z
rotation            =  NO	; Rotate the solute molecule ? (no, yes)
freeze_solute       =  NO	; Freeze the solute molecule ? (no, yes)
seed                =  1993	; Random number generator seed
