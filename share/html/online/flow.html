GROMACS Flow chart

This is a flow chart of a typical GROMACS MD run of a protein in a box of water. A more detailed example is available in the Getting Started section. Several steps of energy minimization may be nescessary, these consist of cycles: grompp -> mdrun.

eiwit.pdb
Generate GROMACS topology  pdb2gmx 
conf.gro topol.top




Enlarge the box   editconf  
conf.gro
Solvate protein  genbox 
conf.gro topol.top
grompp.mdp
Generate mdrun input file  grompp 
topol.tpr traj.trr

Run the simulation (EM, MD or LD)  mdrun 
traj.xtc ener.edr
Analysis  g_... or ngmx   g_energy
